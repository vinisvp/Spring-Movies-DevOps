# Spring-Movies-DevOps

## Resumo
Uma API Rest simples de filmes, permitndo o cadastro de filme, e a consulta de todo o catalogo. Além disso, esse sistema também tem o gerênciamento de contas de usuário e autenticação.

## Requisitos
- Java JDK (versão >= 21)

## Como rodar
1. Clone esse repositório, e acesse o diretório do projeto:
```bash
cd Spring-Movies-DevOps
```
2. No diretório do projeto, inicie o projeto:
```bash
mvnw spring-boot:run
```
